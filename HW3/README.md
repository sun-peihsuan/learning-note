# Binary search tree
## <原理>

  二元搜尋樹（Binary Search Tree）、有序二元樹（ordered binary tree）或排序二元樹（sorted binary tree）

  是指一棵空樹或者具有下列性質的二元樹：

  1.若任意節點的左子樹不空，則左子樹上所有節點的值均小於它的根節點的值；
  
  2.若任意節點的右子樹不空，則右子樹上所有節點的值均大於它的根節點的值；
  
  3.任意節點的左、右子樹也分別為二元搜尋樹；
  
  4.沒有鍵值相等的節點。
