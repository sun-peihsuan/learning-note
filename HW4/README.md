# Hash_table

<原理>

   是根據Key而直接查詢在內存存儲位置的資料結構

   也就是說，它通過計算一個關於鍵值的函數，將所需查詢的數據映射到表中一個位置來查詢記錄，這加快了查找速度。

   這個映射函數稱做Hash fuction，存放記錄的數組稱做Hash table。
