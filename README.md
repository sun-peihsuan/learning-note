WEEK 4	  Insertion sort(插入排序作法)

	將資料分成已排序、未排序兩部份
	依序由未排序中的第一筆(正處理的值)，插入到已排序中的適當位置
	插入時由右而左比較，直到遇到第一個比正處理的值小的值，再插入
	比較時，若遇到的值比正處理的值大或相等，則將值往右移
